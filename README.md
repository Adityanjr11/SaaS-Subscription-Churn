## SaaS Subscription Churn Analysis

### Overview
Conducted churn risk and revenue stability analysis using simulated SaaS subscription data (8K users). Performed cohort retention analysis, engagement segmentation, and revenue-weighted churn investigation to identify high-risk customer segments. Recommended targeted retention strategies projected to reduce churn by 3â€“4%.

### Dataset
- 8,000 simulated users  
- Plan tiers: Basic, Pro, Premium  
- Engagement metrics: Weekly logins, Feature usage, Support tickets  
- Revenue variables: Monthly price, Discounts, Churn status  

### Key Insights
- Overall churn rate: 21.3%  
- Medium-engagement users drive 45% of churn-driven revenue loss  
- Pro tier contributes the highest financial leakage  
- Engagement strongly correlates with churn risk  

### Strategic Recommendations
- Targeted retention incentives for medium-engagement Pro users  
- Expand seat-sharing to increase engagement and switching costs  
- Feature adoption nudges for moderately active users  

### Tools Used
- Python (Data simulation)  
- MySQL (Analysis)  
- SQL (Segmentation and revenue-weighted churn analysis)
